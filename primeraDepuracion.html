Demo: tu primera depuración de código JavaScript

En este primer ejemplo puedes ver cómo incluir un archivo js externo en nuestra web.

Pulsa el botón derecho de tu ratón en esta web y "Ver código fuente de esta página", ahora fíjate cómo la etiqueta "script" hace referencia la archivo "primer-debug-javascript.js", como si se tratara de un css externo.

Pulsa encima de "primer-javascript.js" y verás tu pimer código JavaScript.

Esta primera web pretende sumar dos números. Pero si la pruebas un poco verás que, en lugar de eso, los concatena.

Intenta analizar cómo comprueba que se ha pulsado el botón Suma, cómo recoge el primer número y el segundo y cómo pinta la solución.

No te asustes, no hace falta que comprendas aún este código sólamente que te vayas familiarizando con el.

Intenta descargar el archivo "index.html", junto con el archivo "primer-javascript.js" en tu escitorio e intenta que funcione, por ahora basta con eso.

En el siguiente apartado aprenderás a depurar este código, para comprenter cómo arreglarlo y que sume los números en lugar de concatenarlos.
